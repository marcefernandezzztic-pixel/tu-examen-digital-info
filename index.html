🔒 Sistema anti-trampa

Cada intento de trampa
queda registrado y descuenta puntos.

Si el alumno intenta abrir WhatsApp, Instagram, el buscador o cualquier otra aplicación mientras rinde, el sistema lo detecta al instante.

  • 👁️ Detección inmediata — el sistema monitorea en tiempo real si el alumno sale de la pantalla del examen.
  • 🔔 Alerta sonora — se activa un tono de advertencia audible para el docente cada vez que hay una incidencia.
  • 📋 Registro de faltas — cada salida queda anotada como falta en el informe de resultados con hora y número de intento.
  • 📉 Descuento automático de puntos — las faltas se descuentan de la nota final. El alumno lo sabe antes de empezar.
  • 🆔 Bloqueo por DNI — ningún alumno puede rendir dos veces el mismo examen en la misma sesión.
